Después de configurar todo el entorno de desarollo en una nueva máquina, me doy cuenta que en el Microsoft SQL Management Studio 2008 R2 no funciona el Intellisense

La opción esta marcada y incluso desinstalar e instalar no soluciona el problema

Finalmente la solución como siempre en stackoverflow

Al parecer es un issue ya conocido, al instalar SP1 al Visual Studio el Intellsense del Management Studio deja de funcionar

Afortunadamente tiene solución, bajar esta actualización para SQL Server 2008 R2, en concreto el paquete:


SQLServer2008R2_RTM_CU7_2507770_10_50_1777_x64
o
SQLServer2008R2_RTM_CU7_2507770_10_50_1777_x86


Una vez instalado, Intellisense funciona a la perfección

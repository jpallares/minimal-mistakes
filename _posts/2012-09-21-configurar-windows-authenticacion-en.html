Si instalas IIS con la configuración por defecto esta no incluye Autenticación de Windows

Te puedes volver loco intentando configurar esa autenticación cuando el problema es básicamente que no esta instalada

Para hacerlo basta con marcar este checkbox dentro de Activar o desactivar características de Windows
